# ShiftPlanning

A gem for interfacing with the [Shift Planning API](https://www.shiftplanning.com/api/)

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'shift-planning'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install shift-planning
